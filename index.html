📈 1. Crime Severity Index

The Crime Severity Index (CSI) measures both the volume and severity of crime in Canada. It provides a more comprehensive picture than simple crime counts by weighting crimes according to their seriousness as determined by actual sentences handed down by courts.

Crime Severity Trends (1998-2024)

Source: Statistics Canada Table 35-10-0026-01

Key Findings:

  • Overall Crime Severity Index shows long-term trends from 1998 to 2024, revealing patterns in total crime severity
  • Violent Crime Severity Index tracks serious violent offenses separately, providing insight into violent crime trends
  • Non-Violent Crime Severity Index captures property crimes and other non-violent offenses
  • Data spans 26 years, allowing for comprehensive analysis of crime trends over time

About the Crime Severity Index

The CSI was developed by Statistics Canada to provide a better measure of police-reported crime than simple crime counts. Each crime is weighted based on the average sentence handed down by criminal courts, with more serious crimes receiving higher weights.

🎯 2. Organized Crime

Analysis of organized crime violations in Canada from 2016 to 2024, tracking both total organized crime incidents and the most prevalent violation categories. This data provides insight into the patterns and trends of organized criminal activity across the country.

Organized Crime Trends (2016-2024)

Source: Statistics Canada Table 35-10-0062-01

Key Findings:

  • Total organized crime violations tracked from 2016-2024, showing year-over-year trends
  • Top 3 most common organized crime categories identified and analyzed separately
  • Trends reveal patterns in organized criminal activity over an 8-year period
  • Data shows both overall organized crime trends and category-specific patterns

About Organized Crime Data

This dataset tracks violations where organized crime involvement has been identified by law enforcement. It includes various categories of criminal activity where organized criminal groups are involved in the commission of the offense.

📖 Methodology

Data Sources

All data is sourced from Statistics Canada's official public tables, accessed via their data API:

  • Crime Severity Index: Table 35-10-0026-01
    Covers 1998-2024, includes overall, violent, and non-violent indices
  • Organized Crime: Table 35-10-0062-01
    Covers 2016-2024, includes specific violation categories

Approach

This analysis uses official Statistics Canada data accessed programmatically through their public API. All data processing, analysis, and visualization code is available in the accompanying Python notebook hosted on GitHub, ensuring full reproducibility and transparency.

Limitations

  • Reported crimes only: Data reflects crimes reported to and recorded by police, not all crimes committed
  • CSI weighting: Crime severity weights are determined by Statistics Canada based on court sentences
  • Organized crime data: Available from 2016 onwards only; earlier data not included in this dataset
  • MVP scope: This report includes 2 of 5 planned datasets; full analysis coming December 1, 2025
  • Geographic scope: Analysis focuses on Canada-wide data; provincial breakdowns may be added in future versions

Reproducibility

All code used to generate this analysis is available in the project's GitHub repository. Anyone can run the Python notebook to reproduce these findings or extend the analysis with additional data.
